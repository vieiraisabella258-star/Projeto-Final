# Projeto-Final
Projeto Final do Curso de JavaScript no Senai de São Bernardo - Almirante Tamandaré

Tema do Projeto: Com a crescente vulnerabilidade digital enfrentada pela população idosa, em um mundo cada vez mais conectado, percebemos que a melhor idade, muitas vezes, é o alvo principal de golpes e fraudes online, especialmente aqueles que envolvem o roubo de dados pessoais e a propagação de links maliciosos. Pensando nisso foi desenvolvido o núcleo seguro, criado para ajudar pessoas que passam por dificuldades como essa.
